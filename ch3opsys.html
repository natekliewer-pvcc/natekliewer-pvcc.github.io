Operating Systems Terminology
TERM DEFINITION
operating system The software that controls the overall operation of a computer.
job queue A storage place for jobs waiting to be executed.
FIFO First-In, First-Out.
interacvtive processing A feature which allows a program being executed to carry on a dialogue with the user.
time-sharing A feature used by operating systems that are designed to provide services to multiple users at the same time.
multiprogramming Seemingly doing multiple jobs at once, while in reality doing each job individually for an interval of time.
application software Programs that perform tasks particular to the machine's utilization.
system software Programs that perform tasks common to computer systems in general.
utility software Programs that extend or customize the capabilities of an operating system.
graphical user interface (GUI) Files and programs are represented to and interacted with by the user through icons.
shell The system communicates to the user via text on a monitor.
kernel The internals of an operating system, containing software for very basic functions.
device driver Software units that communicate with peripheral devices' controllers to carry out their operation.
file manager Maintains records of all files in mass storage, those file permissions, file paths, etc.
memory manager Coordinates the use of the machine's main memory.
virtual memory A larger memory space created by paging, or rotating programs between main memory and mass storage.
boot strapping The process of transferring the operating system from mass storage to main memory every time the machine is turned on.
ROM Read-Only Memory -- cannot be altered, only read from.
deadlock Two or more processes cannot proceed because each is waiting on a resource allocated to another.
spooling Holding data for a later (but more convenient) time.
super user (or administrator) A user with very strong access to the operating system including modifying critical packages, adjusting user permissions, and other tasks that are denied to other users.
multi-factor authentication A process requiring multiple different identification methods to gain access to a system, rather than just a password. For example, a password and a code sent to the user's phone.
